# RuedaFácil - Sistema de Alquiler de Vehículos

## Descripción
Aplicación Java para gestionar alquileres de vehículos, clientes, empleados y categorías. Incluye conexión a base de datos MySQL y procedimientos almacenados.

## ¿Qué se ha usado?
- Java
- MySQL
- JDBC

## Estructura del Proyecto
- `App.java`: Principal
- `ConexionBD.java`: Conexión a base de datos
- `DAO/`: Clases de métodos 
- `DTO/`: Clases de datos
- `PROCEDIMIENTOS_PFINAL.sql`: Scripts SQL con triggers, funciones y procedimientos
